# Affiliate Program

This page explains how AntiDrain partner links work.

Short version:

- any AntiDrain page can be opened through a partner link
- after the page opens, the site remembers the reward address from that link
- the total fee stays the same
- part of the fee can be sent to the saved reward address

## Which link formats work

AntiDrain accepts partner links in both of these forms:

- `affiliateAddress`
- `affiliate_address`

Example:

```text
https://antidrain.me/?affiliateAddress=0x1234...abcd
```

After the page opens, AntiDrain removes that extra part from the visible URL.

## Where the reward address is remembered

AntiDrain keeps the saved reward address in:

- browser storage
- a long-lived site cookie for `antidrain.me`

So the saved partner address can survive reloads, later visits, and moving between the main site and the docs.

Invalid addresses are ignored.

If the saved partner address is the same as the protocol fee address, AntiDrain skips the split and keeps the full fee with the protocol.

## What changes when a partner address is saved

A saved partner address does not change:

- the selected route
- the total quoted service fee
- the rescue recipient
- the execution order

It only changes who receives part of the fee.

## Token rescues and Transfer All Native

Without a saved partner address:

- the full 20% fee share goes to the protocol
- the remaining 80% follows the normal recipient path

With a saved partner address:

- 15% goes to the protocol
- 5% goes to the saved partner address
- the remaining 80% still follows the normal recipient path

This applies to:

- ERC-20 transfer entries
- execution-time `Transfer All Native` sweeps

## Fixed 5 USD network fee

NFT fee units and pure custom TX fee units still cost 5 USD total in the network's native token.

Without a saved partner address:

- the full 5 USD equivalent goes to the protocol

With a saved partner address:

- 3.75 USD equivalent goes to the protocol
- 1.25 USD equivalent goes to the saved partner address

The user still pays the same total fee. Only the receivers change.

## How the main site builds links

The main site header now includes an **Affiliate Program** section.

That section lets you:

- enter the reward address that should receive partner payouts
- generate a partner link
- copy the final link

New links generated by the site use `affiliateAddress`. Older links with `affiliate_address` still work.

## Practical notes

- A partner link can point to the main site, a docs page, or any other AntiDrain page.
- The saved reward address works only in the current browser. It is not an onchain profile.
- Opening a newer partner link can replace the saved address.
- You can also clear the saved address manually from the Affiliate Program section.
