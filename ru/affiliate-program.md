# Партнерская программа

На этой странице разобрано, как в AntiDrain работают партнёрские ссылки.

Коротко:

- любую страницу AntiDrain можно открыть по партнёрской ссылке
- после открытия страницы сайт запоминает адрес для наград из этой ссылки
- общий размер комиссии не меняется
- меняется только то, кому уйдёт часть комиссии

## Какие форматы ссылок работают

AntiDrain принимает партнёрские ссылки в двух вариантах:

- `affiliateAddress`
- `affiliate_address`

Пример:

```text
https://antidrain.me/?affiliateAddress=0x1234...abcd
```

После открытия страницы AntiDrain убирает эту лишнюю часть из видимого адреса.

## Где запоминается адрес для наград

AntiDrain хранит сохранённый адрес для наград в:

- хранилище браузера
- долгоживущем cookie для `antidrain.me`

Поэтому сохранённый партнёрский адрес переживает перезагрузку страницы, будущие визиты и переходы между основным сайтом и docs.

Некорректные адреса игнорируются.

Если сохранённый партнёрский адрес совпадает с адресом комиссии AntiDrain, сайт не делит выплату и оставляет всю комиссию протоколу.

## Что меняется, если партнёрский адрес сохранён

Сохранённый партнёрский адрес не меняет:

- выбранный маршрут
- итоговую комиссию
- получателя спасённых активов
- порядок исполнения

Он меняет только то, кто получает часть комиссии.

## Переводы токенов и Transfer All Native

Если партнёрский адрес не сохранён:

- все 20% комиссии идут протоколу
- оставшиеся 80% идут по обычному пути

Если партнёрский адрес сохранён:

- 15% идут протоколу
- 5% идут на сохранённый партнёрский адрес
- оставшиеся 80% всё так же идут по обычному пути

Это применяется к:

- переводам ERC-20
- sweep-операциям `Transfer All Native` во время отправки

## Фиксированная комиссия 5 USD

Комиссия для NFT и чистых custom TX по-прежнему составляет 5 USD в нативном токене сети.

Если партнёрский адрес не сохранён:

- весь эквивалент 5 USD идёт протоколу

Если партнёрский адрес сохранён:

- эквивалент 3.75 USD идёт протоколу
- эквивалент 1.25 USD идёт на сохранённый партнёрский адрес

Для пользователя общий размер комиссии не меняется. Меняются только получатели.

## Как сайт генерирует ссылку

В верхнем меню сайта есть секция **Партнерская программа**.

В ней можно:

- ввести адрес для наград
- сгенерировать партнёрскую ссылку
- скопировать готовую ссылку

Новые ссылки, которые делает сайт, используют `affiliateAddress`. Более старые ссылки с `affiliate_address` тоже продолжают работать.

## Практические замечания

- Партнёрская ссылка может вести на главную страницу, в docs или на любую другую страницу AntiDrain.
- Сохранённый адрес работает только в текущем браузере. Это не onchain-профиль.
- Если позже открыть новую партнёрскую ссылку, сохранённый адрес можно заменить.
- Его также можно вручную очистить из секции **Партнерская программа**.
