# Кошелёк-донор

Кошелёк-донор — это кошелёк, который отправляет транзакции и платит газ.

Он нужен, чтобы основной или скомпрометированный кошелёк не обязан был иметь нативные монеты для газа.

## Зачем нужен донор

Донор:

- платит gas;
- отправляет подготовленный rescue-сценарий;
- может выполнять EIP-7702 flow;
- может отправлять permit rescue;
- может выводить собственные остатки через Asset Manager.

## Как создать донора

1. Откройте `/workspace`.
2. На первом шаге нажмите создание донора.
3. Сохраните backup.
4. Скопируйте адрес донора.
5. Пополните его только тогда, когда сайт покажет нужную сумму.

## Как импортировать донора

Если у вас уже есть донор:

1. Нажмите import.
2. Вставьте private key или другой поддерживаемый формат.
3. Проверьте адрес.
4. Сохраните backup, если это новый для вас донор.

## Backup

Backup нужен, чтобы восстановить донор позже.

Важно: сайт может начать загрузку файла backup, но браузер не всегда гарантирует, что файл реально сохранён. Проверьте папку Downloads или выбранную папку вручную.

## Show / Hide private key

Кнопка Show показывает приватный ключ. Используйте её только когда рядом нет screen share, записи экрана или посторонних людей.

После reload приватный ключ снова должен быть скрыт.

## Когда пополнять донора

Не нужно пополнять донора сразу большой суммой.

Лучше дождаться шага Fund Donor. Там сайт покажет расчёт нужной суммы.
