# Партнёрская ссылка

Страница `/affiliate` помогает создать ссылку с вашим EVM-адресом для возможного fee sharing.

## Как создать ссылку

1. Откройте `/affiliate`.
2. Введите EVM-адрес для выплат.
3. Скопируйте ссылку.
4. Передайте её пользователю.

## Как это работает

Когда пользователь открывает сайт по вашей ссылке, адрес может сохраниться локально как inviter tag.

Если выбранный платный flow поддерживает fee sharing, этот адрес может получить часть комиссии.

## Важно

Партнёрская ссылка не меняет безопасность rescue-сценария.

Если affiliate address не может принять native payout, execution должен оставаться безопасным, но payout может уйти по fallback-логике.

## Что проверять

Проверяйте, что:

- адрес введён правильно;
- ссылка скопирована полностью;
- пользователь открыл именно вашу ссылку;
- flow действительно поддерживает fee sharing.
